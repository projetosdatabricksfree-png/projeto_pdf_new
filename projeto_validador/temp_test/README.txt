README
------------

This download contains all test files and accompanying documentation to perform internal testing and apply for official verification of preflight applications against the GWG 2015 specifications. BEWARE: as from GWG2015, the 'test files' and 'documentation' set are delivered as seperate downloads on the GWG website. Be sure to download them both! 
The documentation is created and maintained by GWG. GWG is not responsible for any mistakes, misinterpretations or costs as a result of the use of these test files. 

It very important to read all documentation and run all test files before official verification of the preflight tool by GWG. There are costs involved with testing! Costs can increase when several test rounds or retests of the preflight solution are required!

VIGC is an independent organization commissioned by GWG for performing the GWG compliancy testing of preflight applications. For questions or remarks, please contact didier.haazen@vigc.be.

Good luck!
